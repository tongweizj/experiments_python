## 迭代计划

1. v1: 实现每个实体的基础 CRUD
2. v2: 添加关联查询功能
3. v3: 完善筛选、分页等高级功能
4. v4: 添加数据验证和错误处理


## v1: 实现每个实体的基础 CRUD

### 1. 定义数据库

#### 作者 (Author)
- ID (主键)
- 姓名
- 国籍
- 出生日期 (可选)
- 简介 (可选)

#### 书籍 (Book)

- ID (主键)
- 书名
- ISBN
- 出版年份
- 作者ID (外键)
- 出版社ID (外键)
- 分类/标签 (可选)

#### 出版社 (Publisher)

- ID (主键)
- 出版社名称
- 地址
- 联系电话 (可选)
- 成立年份 (可选)

### 2. 基础 CRUD 功能需求

#### 作者管理

- 创建作者
- 获取所有作者列表
- 获取单个作者详情
- 更新作者信息
- 删除作者

#### 书籍管理

- 创建书籍
- 获取所有书籍列表
- 获取单个书籍详情
- 更新书籍信息
- 删除书籍

#### 出版社管理

- 创建出版社
- 获取所有出版社列表
- 获取单个出版社详情
- 更新出版社信息
- 删除出版社

##  v2: 添加关联查询功能

### 1. 关联查询需求

#### 书籍相关关联查询
- 查询书籍时包含作者完整信息
- 查询书籍时包含出版社完整信息
- 根据作者ID查询其所有书籍
- 根据出版社ID查询其所有书籍

#### 作者相关关联查询
- 查询作者时包含其所有书籍列表
- 查询作者时包含其合作出版社信息

#### 出版社相关关联查询
- 查询出版社时包含其所有书籍列表
- 查询出版社时包含其签约作者信息

## v3: 完善筛选、分页等高级功能

### 4. 高级查询需求 (可选)

#### 筛选和搜索

- 按作者姓名搜索书籍
- 按出版社名称搜索书籍
- 按书籍分类筛选
- 按出版年份范围筛选

#### 分页和排序

- 列表接口支持分页
- 支持按不同字段排序

## v4: 添加数据验证和错误处理

### 5. 数据验证需求
- 必填字段验证
- ISBN 格式验证
- 日期格式验证
- 外键关系存在性验证

### 6. 错误处理需求

- 实体不存在时的 404 错误
- 数据验证失败的 422 错误
- 删除有关联数据时的约束错误处理
